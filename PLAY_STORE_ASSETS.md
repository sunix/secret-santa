## Tips for Screenshots

1. **Use actual app on Android device:**
   - Install the TWA app
   - Navigate to key screens
   - Use device screenshot function

2. **Alternatively, use Chrome DevTools:**
   - Open app in Chrome
   - Press F12 for DevTools
   - Toggle device toolbar
   - Select Android device (e.g., Pixel 5)
   - Take screenshots

3. **Image editing:**
   - Ensure proper resolution
   - Add device frame (optional but looks professional)
   - Consider adding brief text overlays explaining features
   - Keep consistent style across all screenshots

4. **Recommended screenshots:**
   - Welcome/empty state with "Add participants" visible
   - Screen with 5-6 participants added
   - Couples management with 1-2 couples added
   - Drawing in progress showing reveal
   - Final results summary
   - Language switcher in action (optional)

## Feature Graphic Design Tips

1. Use Christmas colors: red (#c41e3a), green (#165b33), white, gold
2. Include the app icon prominently
3. Add tagline: "Organize Your Secret Santa" or similar
4. Keep text minimal and readable
5. Consider festive elements: snowflakes, gift boxes, Santa hat
6. Maintain brand consistency with app design

Tools for creating graphics:
- Canva (free templates available)
- Figma (free for personal use)
- GIMP (free and open source)
- Photoshop/Illustrator (if available)
